How to Get Your API Key

Step 1: Go to Google AI Studio

Visit the Google AI Studio website by clicking the button below (it may be blocked on some work/school administered devices):

Open Google AI Studio →

Step 2: Sign In

If you're not already signed in, sign in with your Google account.

Step 3: Get API Key

Once signed in:

  1. Look for the "Get API key" button or section
  2. Click on "Create API key"
  3. Your new API key will be generated
  4. Copy the API key to your clipboard
  5. Save this somewhere you can easily access (not here) in case you need it

Step 4: Add API Key to Chat

  1. Return to the chat interface
  2. Click the ⚙️ Settings button
  3. Paste your API key in the input field
  4. Click Save

Important: Keep your API key secure and never share it with others. If you believe your key has been compromised, return to Google AI Studio to revoke it and generate a new one.
